## 🎯 **About The Project**

Pernah kebingungan menerjemahkan teks ke **banyak bahasa sekaligus**? Tool ini solusinya!

**Multi Language NLP Translator** adalah CLI tool sederhana yang bisa menerjemahkan teks ke **6 bahasa dunia** secara **real-time** menggunakan **Google Translate API**.

### ✨ **Kenapa keren?**
- ⚡ **Sekali input, 6 terjemahan langsung keluar**
- 🎯 **Auto-detect** bahasa sumber (nggak perlu pilih-pilih)
- 🚀 **Super fast** karena pake API Google
- 💻 **Lightweight** - cuma butuh 1 file Python!
